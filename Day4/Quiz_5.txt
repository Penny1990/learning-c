    %c => Einzelnes Zeichen
    %s => Zeichenkette => TF_US_P2
    %f => Kommazahl

    Adressoperator &-Operator

    Bitoperatoren 
    Vergleichsoperatoren (< > <= >= == != )

Wie wird ein char (Character)-Variable notiert in C für den Buchstaben H?

    char zeichen = 'H';

Was stellt der sog. &-Operator dar? 
    
    Adressoperator &-Operator

Was wird in einer Anweisung wie scanf("%d", &x); codiert?

    Kodierung von Eingaben
    Speicherung in den RAM
    Ganzzahl wird in die Variable x gespeichert

Was ist der Anwendungszweck der Funktion fflush(stdin)?

    Leerung des Tastaturpuffers

Für die Eingabe welcher Datentypen wird fflush(stdin) empfohlen?

    Character & Strings
    => Meister Speicherplatz notwendig
    => "Hallo Welt"
    => 'H' = 4 Byte
    => 'a' = 4 Byte
    => ...
